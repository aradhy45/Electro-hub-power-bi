📊 Electro Hub Sales Dashboard (Power BI Project)

### 1 Project Overview 
This project analyzes sales performance for an electronics business using Power BI. 
It provides insights into revenue, customer trends, and product performance.

### 2 Objectives
Analyze sales trends
Identify top-performing products
Understand customer purchasing behavior

### 3 Dataset
File Name: Store Data.xlsx
Source: Self-collected / practice dataset
Format: Excel (.xlsx)

### 4 Tools Used
Power BI
Microsoft Excel
DAX

## 📊 Key Business Analysis & Insights

This dashboard answers important business questions using interactive visuals:

### 1️⃣ Top & Bottom Product Performance
Identified Top 5 and Bottom 5 products based on:
  * Sales
  * Profit
  * Quantity Sold

### 2️⃣ Sales Trend Analysis
 Analyzed how sales vary over time:
  * Daily
  * Monthly
  * Quarterly
  * Yearly

### 3️⃣ Sales vs Profit Relationship

* Visualized correlation between sales and profit
* Helps detect high-revenue but low-profit products

### 4️⃣ Period Comparison Analysis

* Users can compare performance between two custom time periods
* Metrics compared:

  * Sales
  * Profit
  * Quantity Sold

### 5️⃣ Discount Analysis

* Calculated average discount across different discount categories
* Helps evaluate impact of discounts on revenue

### 6️⃣ Order Volume

* Total number of orders tracked to measure business activity

### 7️⃣ Detailed Order-Level Analysis

* Interactive table showing:
  * Sales
  * Profit
  * Discount
  * Net Sales
    
* Fully filterable by:
  * Product
  * Date
  * Customer ID
  * Promotion Category

### 8️⃣ Geographic Sales Analysis

* Visualized sales distribution across different cities
* Helps identify high-performing regions
